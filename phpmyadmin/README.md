# Docker para Portainer

---

O phpmyadmin é uma ferramenta de gerenciamento e administração de bancos de dados mysql

Baseando-se no arquivo example.env neste diretório, crie um arquivo de nome `.env`, adicione o host de sua preferência, usuário e senhas do seu banco de dados. Escolha as portas em que seu banco de dados está disponível. Em seguida, basta acessar o host escolhido na porta escolhida. No exemplo em questão, basta acessar `127.0.0.1:8010`
